# Libraries

Use the versions bundled here. Other versions may cause issues.

Additional dependencies (e.g. `AnimatedGIF`) can be installed via Arduino Library Manager.

| Library | Description |
|---------|-------------|
| `Adafruit_LSM6DS.7z` | IMU driver for LSM6DS3TRC |
| `GFX_Library_for_Arduino.7z` | Display driver for ST7789 |
